## Core Principles Applied

Every screen was designed asking:

1. **Does this reduce friction?** If not, remove it.
2. **Does this feel like hosting?** If yes, reframe it.
3. **Does this require the user to perform?** If yes, simplify.
4. **Does this respect hesitation?** If not, add an exit.
5. **Does this disappear when not needed?** If not, question it.
